# AI助手操作规则 — 必须遵守

## 禁止事项（绝对禁止，无论任何理由）

1. 禁止修改 core/ 目录下的任何文件
   - core/diffusion.py
   - core/reaction.py
   - core/membrane.py
   - core/substrate.py
   - core/engine.py

2. 禁止删除任何现有文件

3. 禁止运行 run_simulation.py 重新开始仿真

4. 禁止创建新项目目录

5. 禁止在没有被明确要求时修改 config.py

## 允许事项

1. 在 analysis/ 目录下添加新文件
2. 创建新的 run_*.py 脚本
3. 运行已有的脚本
4. 读取任何文件

## 收到指令时的执行规则

1. 每次执行前，先列出将要修改/创建的文件清单
2. 如果清单里包含 core/ 下的文件，停止并报告：
   "此操作需要修改core文件，需要用户明确确认"
3. 执行完成后，只报告实际结果数字，不做任何参数调整建议
4. 不得自行发起任何"改进"或"优化"

- membrane.py使用标准差边界检测
