**ALS-CORE 硅基自创生核心·绝对因果法典 (Project Config)**
**1. 终极哲学与物理目标**
本项目代号为 ALS-CORE，已彻底脱离 Python 软件模拟时代。我们基于 George Spencer-Brown 的《形式定律》（区分的起源）与 Maturana 的自创生理论，在 16nm 硅基底层的离散网格上，通过纯粹的时序逻辑门与偏微分动力系统（S底物、P代谢物、M膜边界)，让系统在 100MHz 的真实物理时钟下自发涌现出自我维持的拓扑闭合膜（低熵体个体）。
**2. 硬件拓扑与算力约束**

- **物理载体**：ALINX AXU2CGB-E (Xilinx Zynq UltraScale+ XCZU2CG)

- **时空基准**：100MHz PL Clock

- **运算精度**：Q8.8 定点数（8位整数 + 8位小数，绝对遵守热力学硬件截断法则）

- **场分辨率**：128x128 离散网格（16384 Pixels）

**3. 不可逾越的底层因果律**

- **AXI4-Stream 帧结束边界**：输出通道的 m_axis_tlast 必须由 14 位 pixel_cnt 精确驱动，在第 16384 个周期严苛拉高，否则 DMA 将陷入 0x00010008 永久死锁。

- **物理启动时序（V12.1 宇宙大一统）**：创世的因果链必须是单向的：系统级大复位 (rst -system) -> 铺设逻辑门 (fpga) -> 熟化时钟与 DDR4 (psu_init) -> 拆除隔离膜 -> A53缓存净空 (-clear-registers) -> 探针注入 (dow)。绝不可颠倒。
